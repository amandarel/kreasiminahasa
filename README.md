**Project Magang: Website Kreasi Minahasa**

Halo semua, website ini merupakan proyek magang saya di Semester 5, website ini bertujuan untuk mempromosikan Ekonomi Kreatif yang ada di Kabupaten Minahasa.

**Teknologi yang digunakan**

Proyek ini dibangun menggunakan kombinasi teknologi berikut:
- Backend: Flask (Python Framework)
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Database: MySQL
